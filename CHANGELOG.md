## [0.1.0] - 2024-01-01

### Added
- Initial release of @karlorz/image-annotate
- Migrated from vite-component-library-template to image annotation library
- Basic component structure following Atomic Design pattern
- React 19 and TypeScript support
- Vite build configuration for ES and UMD module formats
- Tailwind CSS 4 styling system
- Storybook for component development and documentation
- Vitest for unit testing with React Testing Library
- GitHub Actions workflows for CI/CD
- Release Please integration for automated versioning and changelog generation

### Changed
- Migrated from pnpm to npm for package management
- Updated package name to scoped package @karlorz/image-annotate
- Configured for npm publishing with proper exports field

### Infrastructure
- Added comprehensive GitHub Actions workflows
- Configured Release Please for semantic versioning
- Set up automated npm publishing on releases
- Added .npmignore for optimized package distribution

### Documentation
- Created comprehensive README with installation and usage instructions
- Added CLAUDE.md with development guidelines and important rules
- Established component roadmap for image annotation features

### Upcoming Features (Roadmap)
- ImageCanvas - Base canvas component for displaying images
- BoundingBoxTool - Tool for drawing rectangular annotations
- PolygonTool - Tool for drawing polygon annotations
- PointTool - Tool for marking point annotations
- SelectionTool - Tool for selecting and editing existing annotations
- ZoomPanControls - Controls for zooming and panning the canvas
- AnnotationList - Component to display and manage annotation list
- LabelSelector - Component for selecting annotation labels/classes
- ExportPanel - Component for exporting annotations in various formats (COCO, YOLO, custom JSON)
- ImageAnnotator - Main wrapper component combining all tools
